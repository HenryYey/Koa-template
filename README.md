# Koa项目模板

##### 博客教程：https://blog.csdn.net/weixin_43363255/article/details/88900123

### 这个模板是基于Koa后端项目和Vue前端的，需要修改一下，增添内容才可以运行

## Project setup
```
npm install
```

# 运行后端
```
npm start

```
# 运行前端
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
